  • Задача

    Разработка сервиса, максимально удобного для создания онлайн-интерьера самим пользователем, который сможет самостоятельно создавать желаемый вид и обстановку каждой комнаты.

    Площадка должна позволить оценить стоимость ремонта в зависимости от метража, типа материалов и других параметров, необходимых для точного расчета.

    Создать сервис, позволяющий значительно упростить и сократить время работы проектного персонала, который должен учесть при сборе информации все требования клиентов.
